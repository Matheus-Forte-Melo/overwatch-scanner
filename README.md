# C17 Citizenship Scanner - Introdução

Este é um projeto de uma tela interativa inspirada no jogo "Half-Life 2", um dos meus favoritos! 
O jogo se passa numa distopia onde um experiemento científico deu muito errado e um portal
acabou sendo aberto para a borda das dimensões, Xen. Essa abertura foi rapidamente contida, mas o estrago
chamou a atenção de um império interdimensional, uma União Universal de diversas formas de vida alienginas
que os próprios conquistaram.

Os humanos os chamam de os "Combine". Essa espécie, ou melhor, coletânea de espécies adaptadas
bio-tecnologiamente para atuarem nos ambientes que são melhores designados conquistaram a terra
em cerca de 7 horas durante a "Guerra das 7 Horas". Sim, em menos de 7 míseras horas o planeta
já havia se rendido, sobre ordem do "Administrador" do experiemento que causou tudo isso. Wallace Breen

Alguns os ve com o salvador da humanidade, mas pior, ele assinou a sentença de escrividão do povo humano
por essa União Universal. E assim, a terra se tornou só mais um dos diversos planetas conquistados, e a raça
humana só mais um brinquedo, tecnologicamente amplicado, por meio de práticas horriveis para servir
os ideias expansionistas dos "Combine".

A terra está cada vez mais escassa de recursos por conta da exploração deste império. Os humanos
restantes, escravizados e postos sob uma espécie de ditadura de trabalho forçado e vigilância.
Uma grande inteligência artificial comanda as tropas dos Combine, o "Overwatch".

Este app visa replicar a interface de um localizador de "Anti-cidadões" na área. Tomei a criatividade
de criar algo novo, que nunca apareceu no jogo, mas que faria sentido dado a estética dos softwares
que vemos no titulo mais recente da série, "Half-Life: Alyx".

* * * 

## Features

* Botão de SCAN que exibe aleatoriamente uma mensagem.
* Barra de perigo que aumenta aleatoriamente até entrar em um estado de alerta
* Botão de RESET para barra de perigo

### Interface

Simula um terminal de controle do Jogo Half-Life 2, com uma vibe tecno-brutalista, inspirando-se nos 
softwares dos Combine que conseguimos ver na franquia Half-Life. A interface é a de um Scanner.
