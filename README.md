# C17 Citizenship Scanner - Introdução

Este é um projeto de uma tela interativa inspirada no jogo "Half-Life 2", um dos meus favoritos! O jogo se passa em uma distopia onde um experimento científico deu muito errado e um portal acabou sendo aberto para a borda das dimensões, Xen. Essa abertura foi rapidamente contida, mas o estrago chamou a atenção de um império interdimensional, uma União Universal de diversas formas de vida alienígenas que eles próprios conquistaram.

Os humanos os chamam de "Combine". Essa espécie, ou melhor, coletânea de espécies adaptadas biotecnologicamente para atuarem nos ambientes para os quais são melhor designadas, conquistaram a Terra em cerca de 7 horas durante a "Guerra das 7 Horas". Sim, em menos de 7 míseras horas o planeta já havia se rendido, sob ordem do "Administrador" do experimento que causou tudo isso: Wallace Breen.

Alguns o veem como o salvador da humanidade, mas, pior, ele assinou a sentença de escravidão do povo humano para essa União Universal. E assim, a Terra se tornou só mais um dos diversos planetas conquistados, e a raça humana, apenas mais um recurso, tecnologicamente ampliado por meio de práticas horríveis, para servir os ideais expansionistas dos "Combine".

A Terra está cada vez mais escassa de recursos por conta da exploração desse império. Os humanos restantes estão escravizados e submetidos a uma espécie de ditadura de trabalho forçado e vigilância. Uma grande inteligência artificial comanda as tropas dos Combine: o "Overwatch".

Este app visa replicar a interface de um localizador de "Anticidadãos" na área. Tomei a liberdade criativa de desenvolver algo novo, que nunca apareceu no jogo, mas que faria sentido dado a estética dos softwares que vemos no título mais recente da série, "Half-Life: Alyx".

* * * 

## Features

* Botão de SCAN que exibe aleatoriamente uma mensagem.
* Barra de perigo que aumenta aleatoriamente até entrar em um estado de alerta
* Botão de RESET para barra de perigo
